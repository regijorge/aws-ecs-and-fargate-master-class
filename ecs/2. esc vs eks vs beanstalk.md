# ECS vs EKS vs Beanstalk

## ECS
* Docker on AWS
* Proprietary from AWS
* Good integrations
* EC2 / Fargate launch type models

## EKS
* Docker on AWS
* Running on Kubernets
* Open Sourcing tooling
* EC2 / Fargate launch type models

## Beanstalk
* PasS on AWS
* Run JAVA, Nodejs, Python, C#
* Can leverage ECS
* Less customization
